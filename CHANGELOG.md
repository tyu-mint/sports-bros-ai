# Changelog

## v0.4-editable-context

- 增加消息编辑功能
- 支持编辑自己的消息
- 支持编辑体育生消息
- 修改后直接影响上下文
- 支持删除消息
- 编辑过的消息显示“已修改”

## v0.3-memory

- 增加每个体育生独立长期记忆
- 增加全局记忆：用户画像、偏好、边界、社交观察
- 后台自动记忆提取
- 记忆不显式展示给用户

## v0.2-im-home

- 改成 IM 首页结构
- 置顶群聊
- 5 个体育生私聊入口
- 每个体育生独立 agent prompt
- 教练仅在需要时显示

## v0.1-single-html-chat

- 单 HTML 版本
- 首次输入 Endpoint / API Key / Model
- 体育生兄弟群聊天
- 教练 Agent
- 手机端适配

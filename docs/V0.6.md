# v0.6 Implementation Notes

## 新入口

`index-v060.html`

## 已实现

### Prompt / Agent

- 5 个 agent 年龄统一为 18 岁成年人。
- 加入静态角色卡骨架：核心性格、说话方式、粗口规则、主动消息风格、边界。
- Prompt 中加入三层记忆结构：
  - 用户记忆
  - Agent 角色档案
  - 关系记忆
- 群聊和私聊都传入当前本地时间和时间段。

### 粗口 / NSFW 自动状态

状态：

```text
safe / coarse / suggestive / nsfw
```

实现方式：

- 本地关键词路由作为 fallback。
- 可配置 Router Model 做轻量路由。
- 设置里保留成人/粗口总开关。
- 用户说“正常点 / 别聊黄 / 收一收”等会回到 safe。

### UI 时间

- 每条消息保存完整时间戳。
- 消息间隔超过 5 分钟插入时间分隔条。
- 每条消息底部显示 HH:mm。
- 发给模型的上下文使用时间线格式：

```text
[2026-04-27 23:18] 用户：今天累死了
```

### 主动消息

- 第一版只在网页打开时触发。
- 页面打开后定时检查。
- 支持手动点“主动一下”。
- 根据当前时间段选择候选 agent：
  - 深夜：林川
  - 饭点：赵恺
  - 晚上：周野
  - 其他：陈骁 / 阿航 / 周野
- 可请求浏览器通知权限。

### 轻量路由 / 深思

配置字段：

- Main Model
- Router Model
- Deep Model

路由模型职责：

- 判断是否触发教练。
- 判断是否进入 coarse / suggestive / nsfw。
- 判断是否需要 deep model。

Deep Model 留空时回退到 Main Model。
